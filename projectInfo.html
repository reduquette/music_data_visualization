Data Processing

Once we found our data sets from Kaggle on Rolling Stone Top 500 Songs of All Time and Spotify Tracks DB, we had to process the data using python to combine the data sets. We used the python package pandas to read the csv files and then used the re package to remove any special characters. The songs were listed with different capitalization, spacing, and wording between the data sets to reference the same song. So, we took out spaces, made all letters lowercase, and checked for the word "remastered" appended to the end of each song to identify 442 song matches common to the two data sets.
